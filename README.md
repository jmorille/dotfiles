# My dotfiles

## dotfiles

These are my dotfiles.

## install

```sh
git clone https://github.com/vdupain/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./bin/bootstrap.sh
```
This will symlink the appropriate files in `.dotfiles` to home directory.
Everything is configured and tweaked within `~/.dotfiles`.
